## Why Use GAIA on Messaging Platforms?

You already spend time in Discord and Telegram. Instead of switching to another app, bring GAIA into the conversations you're already having. Manage todos, run workflows, and chat with your AI assistant — all without leaving the platform.

Both bots share the same GAIA backend, so your conversations, todos, and integrations stay in sync across every platform.

## Getting Started

The setup process is nearly identical on both platforms: link your account, then start chatting.

<Steps>
  <Step title="Open a chat with GAIA">
    **Discord:** Add the bot to your server via [this link](https://heygaia.io/discord-bot), or join the [GAIA community server](https://discord.heygaia.io). You can also DM the bot directly.

    **Telegram:** Search for [@heygaia_bot](https://t.me/heygaia_bot) and open a private chat. Send `/start` to begin.
  </Step>

  <Step title="Link your GAIA account">
    Run the `/auth` command on either platform. You will receive a link — click it, sign in to your GAIA account, and your messaging identity is connected automatically.

    ```
    /auth
    ```

    Once linked, GAIA remembers your preferences, integrations, and conversation history across all platforms.
  </Step>

  <Step title="Send your first message">
    Use the `/gaia` command followed by your message:

    ```
    /gaia What's on my schedule today?
    ```

    Or just type freely in a DM — both bots forward plain-text messages to GAIA without needing a command prefix.
  </Step>
</Steps>

<Tip>
  DMs are the most private way to use GAIA on either platform. In a DM, only you and the bot see the conversation.
</Tip>

## Core Features

Both platforms give you access to the same set of capabilities.

### Chat

Send GAIA any question, request, or idea. Responses stream in real time with a typing indicator so you know GAIA is working on it.

```
/gaia Summarize my pending todos and flag anything overdue
```

In DMs, skip the `/gaia` prefix entirely — just type your message.

### Todos

Create, list, complete, and delete todos without opening the web app.

| Command | What it does |
|---|---|
| `/todo list` | Show all active todos |
| `/todo add <title>` | Create a new todo |
| `/todo complete <id>` | Mark a todo as done |
| `/todo delete <id>` | Delete a todo |

### Workflows

Trigger your automations from the chat window.

| Command | What it does |
|---|---|
| `/workflow list` | List all your workflows |
| `/workflow get <id>` | View workflow details |
| `/workflow execute <id>` | Run a workflow manually |

### Conversations

Pick up where you left off or start fresh.

| Command | What it does |
|---|---|
| `/conversations` | Browse your conversation history |
| `/new` | Start a fresh conversation |
| `/stop` | Cancel the current response and reset |

### Utilities

| Command | What it does |
|---|---|
| `/status` | Check if the bot and API are online |
| `/settings` | View your account settings and connected integrations |
| `/help` | Show all available commands |
| `/unlink` | Disconnect your account from GAIA |

<Note>
  For a complete command reference including subcommand options and examples, see the [Bot Commands Reference](/bots/commands).
</Note>

## Discord-Only Features

Discord's richer API unlocks a few extras that are not available on Telegram.

### @Mention in Any Channel

Mention the bot in a server channel and GAIA responds right there — no slash command needed.

```
@GAIA What time is the team standup tomorrow?
```

### Context Menu Actions

Right-click (or long-press on mobile) any message in Discord to access two quick actions:

- **Summarize with GAIA** — Get a 2-3 sentence summary of the selected message.
- **Add as Todo** — Turn any message into a todo item instantly.

### Rich Embeds

Discord responses use native embeds with structured fields, color coding, and action buttons — making lists, settings, and status information easier to scan.

### Ephemeral Replies

Slash command responses are visible only to you by default. Other server members will not see what you asked or what GAIA replied.

### Workflow Creation

Create new workflows directly from Discord:

```
/workflow create <name> <description>
```

<Tip>
  The "Summarize with GAIA" and "Add as Todo" context menu actions work on any message in the server — not just your own. Use them to quickly capture action items from team discussions.
</Tip>

## Telegram-Only Features

### Free-Text Chat in Private Messages

In a private Telegram chat, every message you send goes straight to GAIA. No command prefix, no setup — just type and get a response.

### Command Autocomplete

The bot registers its full command list with Telegram's BotFather, so you get native autocomplete suggestions when you type `/` in the chat.

### Lightweight and Fast

Telegram's bot API keeps things minimal. There is no server to join, no permissions to configure. Open the chat, send `/start`, and you are ready to go.

## Platform Comparison

| Feature | Discord | Telegram |
|---|---|---|
| Slash commands | Yes (with autocomplete) | Yes (with autocomplete) |
| Free-text DMs | Yes | Yes |
| @Mention in groups | Yes | Yes |
| Rich embeds | Yes (native embeds) | No (Markdown fallback) |
| Ephemeral replies | Yes (only you see them) | No (all messages visible in chat) |
| Context menu actions | Yes (Summarize, Add as Todo) | No |
| Workflow creation | Yes (`/workflow create`) | No |
| Server/group required | No (DMs work standalone) | No (private chat only) |

## Tips for Getting the Most Out of GAIA

- **Connect your integrations first.** Link Gmail, Google Calendar, GitHub, and other tools from the [GAIA web app](/guides/connecting-integrations) so GAIA can take action across your stack — no matter which platform you message from.
- **Use DMs for personal tasks.** Keep your todos, calendar queries, and private brainstorms in a direct message with the bot.
- **Use channels for team context.** On Discord, mention GAIA in a project channel so everyone sees the answer. On Telegram, use group @mentions for shared conversations.
- **Start fresh when switching topics.** Run `/new` to clear conversation context before asking about something unrelated. This helps GAIA give more focused answers.

<Warning>
  Your GAIA account must be linked via `/auth` before most commands work. If a command returns an authentication error, run `/auth` and complete the sign-in flow.
</Warning>

## Troubleshooting

| Problem | Solution |
|---|---|
| Bot does not respond | Check that the bot is online with `/status`. On Discord, verify the bot has **Send Messages** and **Use Slash Commands** permissions in your channel. |
| Authentication link expired | Run `/auth` again to generate a fresh link. |
| Commands missing from autocomplete | On Discord, the server admin may need to re-deploy commands. On Telegram, send `/start` to refresh. |
| Long responses are cut off | Discord truncates messages at 2,000 characters. GAIA splits longer responses into multiple messages automatically. Telegram has a 4,096-character limit per message. |
| Markdown looks broken on Telegram | Some advanced formatting is not supported. GAIA falls back to plain text when Telegram cannot parse the formatting. |
