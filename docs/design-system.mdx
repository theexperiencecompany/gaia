## Philosophy

GAIA's UI is **dark-first**, **flat**, and **single-accent**. Every decision flows from five constraints:

- **Dark-first.** The primary experience is dark mode (`#111111` background). Light mode is supported via CSS variables but is secondary.
- **Flat depth.** Depth comes from layered backgrounds (`zinc-800` → `zinc-900`) only. Never borders, rings, or outlines.
- **Single accent.** One primary action color: `#00bbff`. Everything else is zinc-scale neutrals.
- **Borderless cards.** Data cards use background-only separation — no `border-`, `ring-`, or `outline-` in the card tree.
- **Subtle motion.** Animations serve entrance, exit, and state changes. Never decorative. Keep durations ≤ 300ms.

| Context | Class | Size |
|---------|-------|------|

## Dark Card System

Every data card, tool section, and info panel follows this contract exactly.

### Structure

```tsx
const statusClasses = {
  success: "bg-emerald-400/10 text-emerald-400",
  error:   "bg-red-400/10 text-red-400",
  warning: "bg-amber-400/10 text-amber-400",
  info:    "bg-blue-400/10 text-blue-400",
  pending: "bg-zinc-700/50 text-zinc-400",
} as const;

export function MyCard({ title, items, badge }) {
  return (
    <div className="rounded-2xl bg-zinc-800 p-4 w-fit min-w-[400px]">
      <div className="flex items-center justify-between mb-3">
        <p className="text-sm font-semibold text-zinc-100">{title}</p>
        {badge && (
          <span className="rounded-full bg-zinc-700/50 px-2 py-0.5 text-xs text-zinc-400">
            {badge}
          </span>
        )}
      </div>
      <div className="space-y-2">
        {items.map((item) => (
          <div key={item.id} className="rounded-2xl bg-zinc-900 p-3">
            <div className="flex items-center justify-between gap-2">
              <span className="text-sm font-medium text-zinc-200">{item.label}</span>
              <span className={`rounded-full px-2 py-0.5 text-xs ${statusClasses[item.status]}`}>
                {item.value}
              </span>
            </div>
            {item.meta && <p className="text-xs text-zinc-500 mt-1">{item.meta}</p>}
          </div>
        ))}
      </div>
    </div>
  );
}
```

## Depth & Elevation

Depth comes from background layering and blur — not box shadows.

| Context | Technique |
|---------|-----------|
| Buttons, inputs | `shadow-xs` |
| Dialogs, sheets | `shadow-lg` |
| Solid dark cards | No shadow — flat |
| Glass dark cards | `bg-zinc-800/40 backdrop-blur-xl` |
| Hover on dark surfaces | `hover:bg-white/5` |
| Search overlays, full-screen modals | `backdrop-blur-2xl` |

## Motion

### Transition defaults

```tsx
// Default — use for everything
<div className="transition-all duration-200">

// Targeted
<div className="transition-colors duration-200">

// Button press
<button className="active:scale-95 transition-all! duration-300">
```

### Available animation classes

| Class | Duration | Use |
|-------|----------|-----|
| `animate-spin` | Infinite | Loading spinners |
| `animate-pulse` | Infinite | Skeleton placeholders |
| `animate-scale-in` | 0.4s bounce | Element entrance |
| `animate-scale-in-blur` | 0.5s bounce | Blurred entrance |
| `animate-shimmer` | 2s linear | Shimmer effect |
| `animate-shake` | 0.7s | Error feedback |
| `animate-accordion-down/up` | 0.2s ease-out | Accordion open/close |

### Framer Motion

Import from `motion/react` — never `framer-motion`.

```typescript
import { AnimatePresence, m } from "motion/react";

<AnimatePresence mode="wait">
  {visible && (
    <m.div
      initial={{ opacity: 0, y: 8 }}
      animate={{ opacity: 1, y: 0 }}
      exit={{ opacity: 0, y: -8 }}
      transition={{ duration: 0.2, ease: "easeOut" }}
    >
      {content}
    </m.div>
  )}
</AnimatePresence>
```

`AnimatePresence` is required for exit animations. Durations: ≤ 300ms for micro-interactions, ≤ 500ms for page-level entrances.

## Icons

All icons from `@icons` (`@theexperiencecompany/gaia-icons/solid-rounded`). Never raw SVGs or Unicode symbols.

```typescript
import { CheckmarkCircle02Icon, Alert01Icon, Copy01Icon } from "@icons";

// In a button
<Button variant="ghost" size="icon">
  <Copy01Icon className="h-4 w-4" />
</Button>

// In a status badge
<Alert01Icon className="text-amber-400" height={17} />

// With hover scale (parent needs className="group")
<Brain02Icon className="transition-all duration-200 group-hover:scale-110" size={24} />
```

| Context | Value |
|---------|-------|
| Inline / badges | `height={17}` |
| Action buttons | `size={16}` |
| Prominent / decorative | `size={24}` |

<Warning>
  Never use Unicode/text symbols in JSX: no `→`, `↗`, `•`, `✓`, `×`, or similar. Always use icon components from `@icons`.
</Warning>

## Notifications

**Sileo** — mounted globally in the app layout. Call the toast function directly. Never add `<Toaster>` or import from `sonner` / `react-hot-toast`.

```typescript
import { toast } from "sonner";

toast.success("File saved");
toast.error("Something went wrong");
toast.warning("Storage almost full");
toast.info("New message received");
```

Toast style: dark fill `#262626`, white title, `rgba(255,255,255,0.75)` description. Action button colors apply automatically by type.

## Rules

<Columns cols={2}>
  <Column>
    **Do**

    - `rounded-2xl` on all outer card containers
    - `zinc-800` outer → `zinc-900` inner for card depth
    - `/10` opacity backgrounds for all status colors

    - `cn()` for all conditional class merging
    - `transition-all duration-200` as the default transition

    - `rounded-lg` on card containers
    - Solid backgrounds for status badges

    - Add `<Toaster>` — already mounted globally
    - CSS variables on dark card surfaces — use zinc directly
    - Set `font-family` inline — use Tailwind token classes
    - Import from `framer-motion` — use `motion/react`
  </Column>
